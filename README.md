# Professional CV (Personal/cv)

This folder contains the professional CV website.

## Local preview
Open `index.html` in a browser.

## Export PDF
Use the `Download as PDF` button or `Ctrl+P` and Save as PDF.

## Target deploy path
If this folder is placed as `cv/` inside `shajidanam72.github.io`, the URL will be:

`https://shajidanam72.github.io/cv/`
